Conectar con Firebase
Sigue estos pasos una sola vez
1
Crea tu proyecto en Firebase
Ve a console.firebase.google.com → clic en "Crear proyecto" → ponle nombre (ej: misventas) → desactiva Google Analytics → clic "Crear proyecto".
2
Crea la base de datos Firestore
En el menú izquierdo ve a Firestore Database → clic "Crear base de datos" → selecciona "Iniciar en modo de prueba" → elige la región más cercana → "Listo".
3
Registra tu app web
En la pantalla principal del proyecto clic en el ícono </> (Web) → ponle un nombre → clic "Registrar app". Verás un bloque de código así:
const firebaseConfig = {
  apiKey: "AIza...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
4
Pega aquí tu configuración
Copia los valores del bloque anterior y pégalos en los campos de abajo:

Tu configuración se guarda en este navegador — no tendrás que repetir este paso.
